To play the game:
1. Open terminal.
2. Clone the repo.
3. Nagavate inside the repo's quadris directory
4. run make
5. run command line: ./quadris
6. use commands in the command line:
  i. right, left, down, drop
  ii. clockwise, counterclockwise
